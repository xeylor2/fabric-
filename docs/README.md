# FEBRIC Documentation

| Location | Contents |
|---|---|
| `architecture/architecture-v2.md` | **Architecture V2 — the approved, permanent blueprint.** |
| `architecture/snapshot-v2-foundation.md` | **Architecture Snapshot** at the `FEBRIC_V2_FOUNDATION` freeze (M4.1): packages, dependencies, ADRs, milestones, workspace. |
| `adr/` | Architecture Decision Records. Every deviation or addition to the blueprint requires an ADR. |
| `milestones/` | Milestone completion reports (M0, M1, …). |
| `*.pdf` (this folder) | Pre-existing product research documents (onboarding, layout/safe-area research). |

Conventions:

- ADRs are numbered `NNNN-title.md` and never deleted — superseded ADRs are marked as such.
- The blueprint is amended only through an approved ADR; code never silently diverges from it.
